= 备份数据库
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]

NOTE: 您可以使用GUI备份数据库。有关如何使用命令行界面的信息，请参见 https://library.netapp.com/ecm/ecm_download_file/ECMP12395422["《 Snap Creator Framework 4.1.2 管理指南》"^]。
