= 在 SAN 环境中执行单文件还原
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
对于 SAN 环境中的单文件还原，您可以使用 SnapDrive 在 Snap Creator 代理主机上挂载 Snapshot 副本。然后，您可以将要还原的文件复制到数据还原目录，并使用 Snap Creator 自定义操作完成还原处理。
