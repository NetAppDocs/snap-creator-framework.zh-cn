= 正在创建保留策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您计划使用策略对象而不是本地保留策略（此策略包含在配置文件中），则可以创建保留策略。

在创建保留策略时，您可以创建备份类型和策略计划。
