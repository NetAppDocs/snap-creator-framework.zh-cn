= 备份和恢复工作流
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 Snap Creator GUI 将此工作流作为备份和恢复过程的参考。

执行这些任务时， Snap Creator 必须正在运行， Snap Creator GUI 必须处于打开状态。如果不是，则可以在 Web 浏览器（"https://IP_address:gui_port"[] 默认情况下，端口为 8443 ），然后使用 Snap Creator GUI 凭据登录。
