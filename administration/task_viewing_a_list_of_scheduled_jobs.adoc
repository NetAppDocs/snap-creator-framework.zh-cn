= 查看已计划作业的列表
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看已计划作业的列表。

. 从 Snap Creator GUI 主菜单中，选择 * 管理 * > * 计划 * 。
. 从计划选项卡中，查看已计划作业的列表。
+
上次运行结果字段显示上次计划作业的状态。字段中的绿色复选标记表示作业已成功运行，红色 "X" 表示出现故障。
