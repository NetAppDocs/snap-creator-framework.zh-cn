= Snap Creator 密码重置
:icons: font
:imagesdir: ../media/


[role="lead"]
如果忘记了 Snap Creator 密码，可以重置密码。

重置 Snap Creator 密码，

. 导航到 scServer/enginer/etc 文件夹。
. 编辑 snapcreator.properties 文件。
. 在 SNAPCREATOR_Ppass 参数中输入新密码。
+

NOTE: 您可以以纯文本格式提供密码。密码会自动加密。

. 单击 * 保存 * 。
