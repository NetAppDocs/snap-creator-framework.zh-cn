= 查看保留策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看保留策略列表。

. 从 Snap Creator GUI 主菜单中，选择 * 策略 * > * 策略管理 * 。
. 在策略管理器选项卡中，查看策略列表。
